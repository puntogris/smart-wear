# what do i wear
App that will analize the current weather and forecast for the next hours and give you clothing recommendations.

Used in this project:</br>
  * Kotlin
  * MVVM
  * OkHttp
  * Gson
  * Geocoder
  * Dark Sky API
  * Hilt
